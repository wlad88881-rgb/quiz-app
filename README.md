# Программа для тестирования учеников через QR-код

Полностью бесплатно, без ограничений, работает локально (интернет не нужен).

## Установка (один раз)

1. Установите Node.js (если ещё не установлен): https://nodejs.org — скачайте LTS-версию, установите как обычную программу.
2. Распакуйте архив с программой в удобную папку, например `C:\quiz-app`.
3. Откройте эту папку, зажав Shift, кликните правой кнопкой мыши по пустому месту → «Открыть окно PowerShell здесь» (или «Открыть терминал здесь»).
4. Введите команду и нажмите Enter:
   ```
   npm install
   ```
   Подождите, пока установятся зависимости (нужен интернет только на этом шаге, один раз).

## Запуск

В той же папке выполните:
```
npm start
```

В окне терминала появится адрес вида:
```
Адрес в локальной сети (для QR): http://192.168.1.25:3000
```

Не закрывайте это окно, пока идёт тестирование — сервер работает пока оно открыто.

Откройте браузер и перейдите на `http://localhost:3000` — это ваша панель преподавателя.

## Важно про сеть

- Ваш компьютер и телефоны учеников должны быть подключены **к одной Wi-Fi сети**.
- Если Windows спросит про доступ Node.js к сети (брандмауэр) — разрешите доступ для частных сетей.
- Интернет не требуется вообще — всё работает по локальной сети.

## Как пользоваться

1. На главном экране нажмите **«+ Новый тест»**, впишите название, добавляйте вопросы, варианты ответов и отмечайте галочкой правильный(е) вариант(ы).
2. Сохраните тест.
3. Нажмите **«Начать сессию»** у нужного теста — появится QR-код и короткий код (например `W48DZ`).
4. Покажите QR-код на экране/проекторе — ученики сканируют его камерой телефона (или сами открывают ссылку и вводят код вручную, если нет камеры).
5. Каждый ученик вводит своё имя и проходит тест на своём телефоне.
6. Вы в реальном времени видите на панели, кто подключился и кто уже завершил тест, с баллами.
7. По окончании нажмите **«Завершить сессию»** — новые ответы приниматься не будут.
8. Кнопка **«Экспорт в Excel»** скачает файл `.xlsx` с полными результатами (баллы, проценты, ответы по каждому вопросу).

## Данные

Все тесты и результаты хранятся локально в файле `data/db.json` на вашем компьютере — никуда не отправляются.
Если нужно сделать резервную копию — просто скопируйте файл `data/db.json`.

## Вариант Б: разместить онлайн (работает по мобильному интернету, без Wi-Fi)

Если нужно, чтобы ученики заходили с мобильного интернета из любой сети (не обязательно одна Wi-Fi с вами), программу можно бесплатно разместить на хостинге **Render.com**. Тогда компьютер держать открытым во время урока не нужно — сервер работает сам, 24/7, на одном постоянном адресе.

### Шаг 1. Создать аккаунт на GitHub

1. Зайдите на **github.com**, нажмите **Sign up**, заведите бесплатный аккаунт (нужен только email)

### Шаг 2. Загрузить программу на GitHub

Проще всего через сайт, без установки дополнительных программ:
1. На github.com нажмите **+** (вверху справа) → **New repository**
2. Назовите репозиторий, например `quiz-app`, оставьте **Public** или **Private** (без разницы), нажмите **Create repository**
3. На странице репозитория нажмите **uploading an existing file**
4. Перетащите туда ВСЕ файлы и папки из папки `testapp` (кроме `node_modules`, если она там есть — её загружать не нужно)
5. Внизу нажмите **Commit changes**

### Шаг 3. Создать аккаунт на Render и подключить репозиторий

1. Зайдите на **render.com**, нажмите **Get Started**, зарегистрируйтесь (можно через тот же GitHub-аккаунт — так проще, он сам всё подключит)
2. На панели Render нажмите **New +** → **Web Service**
3. Выберите ваш репозиторий `quiz-app` из списка (если не появился — нажмите «Configure account» и разрешите Render доступ к репозиторию)
4. Заполните настройки:
   - **Name**: любое имя, например `quiz-app` (оно станет частью адреса)
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: выберите **Free**
5. Нажмите **Create Web Service**

Render соберёт и запустит программу автоматически (займёт 2-5 минут). После этого вверху страницы появится ваш постоянный адрес вида:
```
https://quiz-app-xxxx.onrender.com
```

Откройте его в браузере — это и есть ваша панель преподавателя, доступная из любой сети.

### Важные особенности бесплатного тарифа Render

- **Данные могут сбрасываться.** Бесплатный тариф не сохраняет файлы между перезапусками сервера. Практический совет: **после каждой сессии тестирования сразу нажимайте «Экспорт в Excel»**, чтобы результаты не потерялись.
- **Первый заход после паузы медленнее.** Если сервером не пользовались 15+ минут, он «засыпает» — первое открытие страницы после этого займёт 30-50 секунд, дальше работает быстро. Рекомендуется открыть панель за пару минут до урока, чтобы сервер «проснулся».
- Дальше ничего вручную запускать не нужно — просто открывайте ссылку `https://quiz-app-xxxx.onrender.com` в браузере на любом устройстве в любой момент.
